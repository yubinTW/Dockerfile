# node 18 with Cypress 13.0.0

## Environment

Debian 11

## Install Package

- `node` 18.17.1
- `npm` 9.8.1
- `cypress` 13.0.0
- `wget`
- `curl`
- `build-essential`
- `software-properties-common`
- `apt-transport-https`
- `dotnet-runtime-6.0`
- `iputils-ping`
- `dnsutils`
- `unzip`
- `libaio1`
- `libaio-dev`

## Build image

```
docker build -t yubintw/node:18-cypress-13.0.0 .
```
