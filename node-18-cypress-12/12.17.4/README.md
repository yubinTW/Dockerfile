# node 18 with Cypress 12.17.4

## Environment

Debian 11

## Install Package

- `node` 18.16.1
- `npm` 9.7.2
- `cypress` 12.17.4
- `wget`
- `curl`
- `build-essential`
- `software-properties-common`
- `apt-transport-https`
- `dotnet-runtime-6.0`
- `iputils-ping`
- `dnsutils`
- `unzip`
- `libaio1`
- `libaio-dev`

## Build image

```
docker build -t yubintw/node:18-cypress-12.17.4 .
```
