# node 16 with Cypress 11.1.0

## Environment

Debian 10

## Install Package

- `node` 16.18.0
- `npm` 8.19.2
- `cypress` 11.1.0
- `wget`
- `curl`
- `build-essential`
- `software-properties-common`
- `apt-transport-https`
- `dotnet-runtime-6.0`

## Build image

```
docker build -t yubintw/node:16-cypress-11.1.0 .
```
