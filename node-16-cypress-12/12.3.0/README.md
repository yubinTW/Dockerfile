# node 16 with Cypress 12.3.0

## Environment

Debian 10

## Install Package

- `node` 16.18.1
- `npm` 8.19.3
- `cypress` 12.3.0
- `wget`
- `curl`
- `build-essential`
- `software-properties-common`
- `apt-transport-https`
- `dotnet-runtime-6.0`
- `iputils-ping`
- `dnsutils`
- `unzip`
- `libaio1`
- `libaio-dev`

## Build image

```
docker build -t yubintw/node:16-cypress-12.3.0 .
```
